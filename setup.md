# Simple Setup

## 1. Get API Keys

- **Groq API**: https://console.groq.com/
- **Google Sheets**: Create a new sheet

## 2. Import to n8n

1. Open n8n
2. Click "Import from File"
3. Select `workflow.json`
4. Add your API keys when prompted

## 3. Run It

Click "Execute Workflow" - it will start scanning immediately.

## That's it!

The workflow will automatically:
- Find new AI research
- Analyze it with AI  
- Save to your Google Sheet
