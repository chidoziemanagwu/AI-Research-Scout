# AI Research Scout - n8n Workflow

Automatically finds and analyzes the latest AI research from ArXiv and Hacker News.

## What It Does

- Scans ArXiv for new AI papers
- Checks Hacker News for AI discussions  
- Uses AI to summarize findings
- Saves everything to Google Sheets

## Quick Setup

1. **Download** `workflow.json`
2. **Import** into n8n
3. **Add credentials:**
   - Groq API key
   - Google Sheets access
4. **Run** the workflow

## Files

- `workflow.json` - The main n8n workflow
- `setup.md` - Simple setup instructions
